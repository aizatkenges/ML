# Machine Learning Projects

This repository contains machine learning projects developed using Python and Scikit-learn.

## Project Overview

The goal of these projects is to apply machine learning techniques to real-world datasets, including classification, regression, and exploratory data analysis tasks.

## Topics Covered

* Data Cleaning and Preprocessing
* Feature Engineering
* Classification Models
* Regression Models
* Model Evaluation
* Data Visualization

## Algorithms Used

* K-Nearest Neighbors (KNN)
* Logistic Regression
* Linear Regression
* Decision Trees
* Random Forest

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* RMSE
* Cross Validation

## Technologies

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Jupyter Notebook

## Datasets

* Iris Dataset
* Penguins Dataset
* Tips Dataset
* Life Satisfaction Dataset
* Supermarket Dataset

## Learning Outcomes

* Building and evaluating machine learning models
* Data preprocessing and feature engineering
* Model comparison and performance analysis
* Working with real-world datasets
